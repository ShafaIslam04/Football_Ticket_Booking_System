# Football Ticket Booking System
 ## This is a database design for a Football Ticket Booking System. The system allows football fans to purchase tickets for matches, while ticket managers can manage match information and booking records.

 ## This demonstrates:
###
- Database Schema Design
- Primary Key & Foreign Key Relationships
- Entity Relationship Diagram (ERD)
- SQL Queries using JOINs, Subqueries, Aggregation, NULL Handling, Pattern - Matching, and Pagination

## SQL Concepts Used
- SELECT
- WHERE
- AND / OR
- ILIKE
- IS NULL
- COALESCE
- INNER JOIN
- LEFT JOIN
- FOREIGN KEY
- PRIMARY KEY
- Aggregate Functions (AVG)
- Subqueries
- ORDER BY
- OFFSET
- LIMIT
